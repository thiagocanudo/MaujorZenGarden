Orientações gerais

Esta página foi desenvolvida para o “Zen Garden do Maujor”.

Uma proposta desafio neste site, visando a estimular suas pesquisas, estudos, habilidades e interesse pelas CSS.

Você faz o download do código HTML e da CSS do tema desta página.

Construa um outro layout para a página, modificando a vontade o arquivo CSS. Mas, nada, nada mesmo do código HTML deverá ser modificado.

O HTML DA PÁGINA É INTOCÁVEL.

O foco exclusivo do site e consequentemente deste desafio são as Folhas de Estilo. A finalidade deste trabalho em particular, é a de expor o poder das CSS, no controle do layout. Assim, nadinha do HTML pode ser modificado.

Nos temas enviados, o conteúdo de todas as páginas é rigorosamente o mesmo, vale dizer não se muda o HTML. O visual é consequência da mudança nas CSS. Uma CSS para cada tema.

Para participar, você constrói um tema, e envia os arquivos zipados (tema.css e imagens) e eu publico. Mostre sua habilidade com CSS e veja seu trabalho publicado juntamente com um link para seu e-mail ou Site/Blog.

Estrutura do site

O site não usa Tabelas para definir seu layout.

As técnicas CSS preconizam a utilização de tabelas somente para apresentação de dados tabulares. Assim, usar tabelas para montar layout, quer seja para compartimentar uma página em colunas e linhas, quer para posicionar elementos é uma prática que deverá ser abandonada pelo desenvolvedor Web moderno.

Por que CSS

A grande vantagem do uso de CSS é a de separar a marcação HTML, da apresentação do design final. Em outras palavras vale dizer que HTML destina-se unicamente a estruturar o conteúdo ficando por conta das CSS toda a responsabilidade pela apresentação “visual”.

Assim, HTML marca textos, cabeçalhos, parágrafos, links, botões, forms, imagens, etc e CSS define cores, posicionamento na tela, estilos de linhas, bordas e toda a parte visual.

A combinação acima resulta em múltiplas vantagens e benefícios que não são conseguidos quando se utiliza o conceito baseado no uso de tabelas, hoje largamente empregado, para construção de Web sites.

Imagine você projetar um site inteiro com uma marcação HTML única que seja navegável tanto em monitor de 1280x1024 quanto em uma telinha de 320x240 de um PDA. É fantástico e possível. Basta entregar às CSS a missão de acomodar o conteúdo na tela

Algumas restrições ainda freiam o uso pleno geral e irrestrito de CSS na Web. As restrições são por conta da incapacidade dos browsers mais antigos dar suporte pleno à CSS. Com o passar do tempo a tendência é a de cair em desuso total os browsers antigos que não suportam ou mesmo suportam parcialmente CSS. Soma-se a favor da CSS a normatização que o W3C vem implementando sobre o assunto e o desejo dos fabricantes de se adequar àquelas normas. é uma questão de tempo!
